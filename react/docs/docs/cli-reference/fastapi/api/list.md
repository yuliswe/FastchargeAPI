# fastapi api list

List available endpoints of an app.

## Usage

    fastapi api list [APP_NAME]

## Example 

    $ fastapi api list myapp

    ID:            endp_WyJteWFwcCIsMTY3Nzk3MTMyNTU1OF0
    HTTP Method:   GET
    Endpoint:      https://myapp.fastchargeapi.com/example
    Example website
